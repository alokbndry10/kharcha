Kharcha is a modern, high-performance web application designed to streamline accounting and expense tracking. Built with focus on efficiency and scalability, it provides a comprehensive suite of tools for managing vendors, items, and day-to-day business expenses.

> [!NOTE]
> This project was **Developed by Aalok Bhandari, Bidhan Thapaliya, Sidddhant Thapaliya**.

## Key Features

- **Intuitive Dashboard**: Real-time overview of financial metrics and activity.
- **Expense Tracking**: Easily record, categorize, and monitor business expenses.
- **Vendor Management**: Keep a detailed database of suppliers and service providers.
- **Inventory & Units**: Manage items and measurement units with precise control.
- **Secure Authentication**: Robust user management powered by Supabase.
- **Modern UI/UX**: Premium interface built with Ant Design and Glassmorphism aesthetics.

## Technology Stack

- **Frontend**: [React](https://reactjs.org/) & [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Ant Design](https://ant.design/)
- **Backend/Auth**: [Supabase](https://supabase.com/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Routing**: [React Router](https://reactrouter.com/)

## Project Setup

This project uses [pnpm](https://pnpm.io/) as the package manager and [Vite](https://vitejs.dev/) as the build tool.

### Prerequisites

- Node.js (Latest LTS recommended)
- pnpm (recommended) or npm

### Installation

1. Clone the repository and install dependencies:
   ```bash
   pnpm install
   # or
   npm install
   ```

2. Configure environment variables in a `.env` file (see `.env.example`).

### Development

To start the development server:
```bash
pnpm dev
# or
npm run dev
```

### Production

To build the project for production:
```bash
pnpm build
# or
npm run build
```
